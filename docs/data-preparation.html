Data Preparation

Overview

Preparing datasets is a critical step in training the VQA Model. This involves generating synthetic data, adding controlled noise, and preprocessing.

Dataset Generation

The dataset consists of synthetic images created using geometric shapes such as circles, squares, and triangles, combined with questions about the spatial relationships between these shapes.

Incorporating Noise

Noise is added at various levels to evaluate the model's robustness. This includes visual noise in images and semantic noise in questions.

Preprocessing Steps

All input data undergoes preprocessing before it's fed into the model. This includes image resizing, normalization, and question tokenization. 

def preprocess_image():
    return Compose([
        Resize((224, 224)),
        ToTensor(),
        Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
    ])
